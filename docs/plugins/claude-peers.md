# claude-peers

- **Type:** MCP Server
- **Category:** collab
- **Tags:** multi-instance, messaging, coordination

## Key Takeaways

- 多个 Claude Code 实例之间互相通信
- set_summary 让其他实例知道你在做什么
- 收到消息要立即回复，不能等当前任务做完
- 适合多窗口并行开发、code review 协同

## Relevance

- 并行 agent 场景（paper-review 的 4 reviewer）理论上可以用 peers 协调
