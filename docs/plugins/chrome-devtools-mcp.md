# chrome-devtools-mcp

- **Source:** [anthropics/chrome-devtools-mcp](https://github.com/anthropics/chrome-devtools-mcp)
- **Type:** Plugin (MCP)
- **Category:** browser
- **Tags:** devtools, screenshot, lighthouse, a11y, automation

## Key Takeaways

- 通过 Chrome DevTools Protocol 直接控制浏览器
- 前端调试、截图、Lighthouse audit、a11y 检查都能做
- 比 Playwright MCP 更底层，适合精细控制场景
- debug-optimize-lcp skill 专门优化 LCP 性能指标

## Relevance

- 前端 skill 开发时用来验证 UI 输出（HTML 预览、截图对比）
