# Ecosystem: Information Presentation

Tools for creating visual presentations, slides, and polished UI from Claude Code.

## frontend-design (Anthropic Official Plugin)

> https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design

Create distinctive, production-grade frontend interfaces with high design quality. Generates creative, polished code and UI design that avoids generic AI aesthetics.

Use when: building web components, pages, dashboards, React components, HTML/CSS layouts, landing pages, or when styling/beautifying any web UI.

Key feature: iterative design process with visual exploration, helps non-designers discover their aesthetic rather than making abstract choices.

## frontend-slides

> https://github.com/zarazhangrui/frontend-slides

Create stunning, animation-rich HTML presentations from scratch or by converting PowerPoint files.

Use when: building a presentation, converting PPT/PPTX to web, or creating slides for a talk/pitch.

Features:
- Rich animations and transitions
- PowerPoint file conversion
- Visual exploration for design choices
- Production-quality output

## How Boris Cherny's team uses presentations

From Boris Cherny's CC tips (Source 5):
> "Have Claude generate a visual HTML presentation explaining unfamiliar code or concepts."

This is tip #10 from the CC creator -- using Claude Code not just for coding but for learning and communicating through visual presentations.

## Decision Guide

| Need | Tool |
|------|------|
| Web UI, landing pages, dashboards | frontend-design |
| Slide decks, presentations, talks | frontend-slides |
| Explaining code/concepts visually | Either -- generate HTML presentation |
