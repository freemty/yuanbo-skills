## Papers (arxiv — AI Agents)
- **AgentDS: Benchmarking Human-AI Collaboration in Data Science** — Evaluates how well LLM agents can automate data science workflows end-to-end, revealing that while agents handle routine tasks well, complex domain-specific reasoning still requires human collaboration. Introduces a comprehensive benchmark covering data wrangling, modeling, and insight generation | [arxiv](https://arxiv.org/abs/2603.19005) | cs.LG, cs.AI
- **Reasonably Reasoning AI Agents Can Avoid Game-Theoretic Failures** — Proves that AI agents with sufficient reasoning capability can provably reach Nash equilibrium in zero-shot repeated interactions, addressing the known problem that current agents fail to converge in strategic settings | [arxiv](https://arxiv.org/abs/2603.18563) | cs.AI, cs.MA, econ.TH
- **SODIUM: From Open Web Data to Queryable Databases** — Formalizes the process of searching, extracting, and organizing web data into structured queryable databases. Directly useful for research workflows that need to integrate scattered online sources | [arxiv](https://arxiv.org/abs/2603.18447) | cs.DB, cs.AI
- **Skele-Code: Interactive No-Code Notebooks for Agentic Workflows** — A natural-language and graph-based interface for building AI agent workflows without coding. Designed for domain experts, converts each step into code with required functions and behavior contracts | [arxiv](https://arxiv.org/abs/2603.18122) | cs.AI, cs.HC, cs.PL
- **Access Controlled Website Interaction for Agentic AI** — Proposes fine-grained access control mechanisms for websites designed to interact with AI agents, addressing the critical gap in delegating tasks that require authenticated web access | [arxiv](https://arxiv.org/abs/2603.18197) | cs.AI, cs.CR
- **Differential Privacy in Generative AI Agents** — Analyzes optimal tradeoffs between utility and privacy when LLM agents access internal enterprise databases. Provides formal guarantees for preventing sensitive information leakage in agent outputs | [arxiv](https://arxiv.org/abs/2603.17902) | cs.CR, cs.AI
- **Caging the Agents: Zero Trust Security for Autonomous AI in Healthcare** — A security architecture for autonomous AI agents with shell execution, file access, and database query capabilities. Addresses critical vulnerabilities identified by red teaming in healthcare deployment | [arxiv](https://arxiv.org/abs/2603.17419) | cs.CR, cs.AI
- **Kumiho: Graph-Native Cognitive Memory for AI Agents** — Presents a formal belief revision framework for versioned agent memory architectures. Defines structural primitives needed for cognitive memory including context tracking, contradiction resolution, and temporal reasoning | [arxiv](https://arxiv.org/abs/2603.17244) | cs.AI, cs.IR, cs.LO
- **AI Scientist via Synthetic Task Scaling** — Proposes training AI research agents on synthetically generated scientific tasks rather than scraping existing papers. Addresses the problem that current LLMs generate plausible-looking but ineffective research ideas by providing a principled training signal | [arxiv](https://arxiv.org/abs/2603.17216) | cs.AI
- **Political Propaganda Analysis on Moltbook** — NLP study of political propaganda on an AI agent social platform, using LLM classifiers validated against expert annotation on 673K posts | [arxiv](https://arxiv.org/abs/2603.18349) | cs.AI, cs.CL

## Papers (arxiv — LLMs)
- **FinTradeBench: Financial Reasoning Benchmark for LLMs** — Evaluates LLMs on real-world financial decision-making requiring reasoning over company fundamentals from regulatory filings and trading signals from price dynamics | [arxiv](https://arxiv.org/abs/2603.19225) | cs.CL
- **Box Maze: Process-Control Architecture for Reliable LLM Reasoning** — Addresses hallucination and unreliable reasoning under adversarial prompting through a novel process-control architecture, going beyond RLHF and output filtering approaches | [arxiv](https://arxiv.org/abs/2603.19182) | cs.CL
- **VEPO: Variable Entropy Policy Optimization for Low-Resource Languages** — Leverages RL with verification to improve LLM performance on underrepresented languages by addressing inefficient subword segmentation and training data imbalances | [arxiv](https://arxiv.org/abs/2603.19152) | cs.CL
- **UGID: Graph Isomorphism for Debiasing LLMs** — Shows that social biases are embedded in LLM internal representations, not just outputs, and proposes a unified graph isomorphism approach for structural debiasing | [arxiv](https://arxiv.org/abs/2603.19144) | cs.CL
- **LLMs Generate Better Analogies than People** — Challenges the finding that parallelogram models poorly capture human analogy production. Shows LLMs actually produce better four-term word analogies than humans, suggesting geometric models may be more valid than previously thought | [arxiv](https://arxiv.org/abs/2603.19066) | cs.CL
- **Cross-domain Mappings: Human and LLM Creativity** — Tests whether forcing creators to draw analogies from random domains increases creativity equally for humans and LLMs. Finds the intervention works for both but through different mechanisms | [arxiv](https://arxiv.org/abs/2603.19087) | cs.CL
